# 第四章 视觉生成模型

生成模型的目标是学习数据分布 $p_{\text{data}}(\mathbf{x})$，进而从中采样生成新的数据。视觉生成领域近年来取得了惊人的进展——从 2014 年 GAN 生成模糊的人脸，到 2022 年 Stable Diffusion 生成照片级真实的图像，再到 2024 年 Sora 生成连贯的视频。本章系统介绍视觉生成模型的核心技术。

## 生成模型的分类

视觉生成模型可按建模方式分为几大类：

**隐变量模型**：假设数据由隐变量 $\mathbf{z}$ 生成，学习 $p(\mathbf{x} | \mathbf{z})$。

- 变分自编码器（VAE）：通过变分推断学习隐空间
- 生成对抗网络（GAN）：通过对抗训练隐式学习分布

**得分/能量模型**：学习数据分布的得分函数 $\nabla_\mathbf{x} \log p(\mathbf{x})$。

- 扩散模型（Diffusion Model）：通过去噪过程学习得分
- 流匹配（Flow Matching）：学习概率流的速度场

**自回归模型**：将图像视为序列，逐像素/逐 patch 生成。

- PixelCNN、ImageGPT
- 视觉自回归（如 LlamaGen）

本章重点讨论当前最成功的范式：VAE（作为潜空间压缩器）和扩散模型（作为生成器）的组合。

## 本章结构

**VAE**（4.1）介绍变分自编码器的原理，它在现代生成模型中主要作为图像的编码器-解码器，将高维像素空间压缩到低维潜空间。

**GAN**（4.2）回顾生成对抗网络的核心思想。虽然 GAN 在图像生成领域已被扩散模型超越，但其对抗训练的思想仍有广泛应用。

**去噪概率模型**（4.3）是本章的核心。我们将从 DDPM 出发，讨论扩散过程的数学原理、DDIM 的加速采样，以及 Flow Matching 的统一视角。

**扩散模型结构**（4.4）讨论 U-Net 和 DiT（Diffusion Transformer）两种主流架构。

**常用技术**（4.5）介绍条件生成的关键技术：CFG（Classifier-Free Guidance）和 ControlNet。

**视频生成**（4.6）简要讨论从图像生成到视频生成的扩展，包括时序建模和一致性控制。

## 符号约定

| 符号 | 含义 |
|------|------|
| $\mathbf{x}$ | 原始数据（图像） |
| $\mathbf{z}$ | 隐变量 |
| $\mathbf{x}_t$ | 时刻 $t$ 的噪声数据（扩散模型） |
| $\epsilon$ | 高斯噪声，$\epsilon \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$ |
| $\alpha_t, \bar{\alpha}_t$ | 扩散过程的噪声调度参数 |
| $p_\theta$ | 参数化的生成分布 |
| $q$ | 真实数据分布或前向过程分布 |
