# 第五章 强化学习基础

强化学习（Reinforcement Learning, RL）研究智能体如何通过与环境交互来学习最优行为策略。与有监督学习从标注数据中学习不同，强化学习从试错经验中学习——智能体采取行动，观察结果，根据奖励信号调整策略。这种学习范式天然适合序贯决策问题，也是大语言模型与人类偏好对齐（RLHF）的理论基础。

## 从监督学习到强化学习

监督学习假设存在一个"正确答案"作为训练信号。但许多问题没有唯一正确答案：

- 下围棋的最优落子是什么？
- 机器人行走的最佳步态是什么？
- 语言模型应该生成哪种风格的回复？

这些问题的共同特点是：好坏由最终结果决定，而非每一步都有明确标签。强化学习正是处理这类问题的框架。

## 核心概念速览

**马尔可夫决策过程**（Markov Decision Process, MDP）是强化学习的数学框架，由五元组 $(S, A, P, R, \gamma)$ 定义：

- $S$：状态空间
- $A$：动作空间
- $P(s' | s, a)$：状态转移概率
- $R(s, a)$：奖励函数
- $\gamma \in [0, 1)$：折扣因子

**策略** $\pi(a|s)$ 定义了在状态 $s$ 下选择动作 $a$ 的概率分布。强化学习的目标是找到最优策略 $\pi^*$，最大化期望累积奖励。

**价值函数**量化了一个状态或状态-动作对的长期价值：

$$V^\pi(s) = \mathbb{E}_\pi \left[ \sum_{t=0}^\infty \gamma^t R(s_t, a_t) \mid s_0 = s \right]$$

$$Q^\pi(s, a) = \mathbb{E}_\pi \left[ \sum_{t=0}^\infty \gamma^t R(s_t, a_t) \mid s_0 = s, a_0 = a \right]$$

## 章节结构

本章从策略梯度和价值函数的基础理论出发，介绍广义优势估计（GAE）这一方差缩减技术，辨析 value-based 和 policy-based 两类方法的特点。随后分别讨论 Off-policy 和 On-policy 优化算法：

- **Off-policy**：Q-Learning、DPO、AlphaZero 等，数据可重复使用
- **On-policy**：PPO、GRPO、DAPO 等，需要实时采样

大语言模型的对齐训练（RLHF）将这些算法应用于文本生成场景，其中 PPO 和 DPO 是两种主流方案，GRPO 则是专为 LLM 设计的新方法。

## 符号约定

| 符号 | 含义 |
|------|------|
| $\pi_\theta$ | 参数为 $\theta$ 的策略 |
| $\pi_{\text{ref}}$ | 参考策略（通常是 SFT 模型） |
| $V(s)$ | 状态价值函数 |
| $Q(s,a)$ | 动作价值函数 |
| $A(s,a)$ | 优势函数，$A = Q - V$ |
| $r_t$ | 时刻 $t$ 的即时奖励 |
| $\gamma$ | 折扣因子 |
| $\lambda$ | GAE 中的 trace 衰减参数 |
