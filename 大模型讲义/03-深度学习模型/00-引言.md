# 第三章 深度学习模型

深度学习的核心在于通过多层非线性变换，将原始输入逐步映射到抽象的语义表示。本章从神经网络的基础结构出发，逐步构建起理解现代大语言模型所需的技术栈。

## 历史脉络

神经网络的发展经历了三次浪潮。1943 年 McCulloch 和 Pitts 提出了形式神经元模型；1958 年 Rosenblatt 发明感知机；1969 年 Minsky 和 Papert 证明感知机无法处理 XOR 问题，神经网络研究陷入第一次低谷。

1986 年 Rumelhart、Hinton 和 Williams 重新发现反向传播算法，多层感知机（MLP）成为可能。但受限于计算资源和数据规模，神经网络在 1990 年代被支持向量机等"浅层"方法超越，进入第二次低谷。

2006 年 Hinton 提出深度信念网络（DBN）的逐层预训练方法，标志着深度学习的复兴。2012 年 AlexNet 在 ImageNet 竞赛中大幅领先传统方法，深度学习进入爆发期。2017 年 Vaswani 等人提出 Transformer 架构，彻底改变了序列建模的范式。

## 本章结构

本章内容按照从简单到复杂、从基础到前沿的顺序组织：

**经典网络结构**（3.1-3.2）介绍深度神经网络（DNN）、卷积神经网络（CNN）、循环神经网络（LSTM）以及多层感知机（MLP）。这些是理解后续内容的基础。

**多模态基础**（3.3）以 CLIP 为例，讨论如何将不同模态（如图像和文本）映射到共享的语义空间。对比学习的思想对后续理解多模态大模型至关重要。

**Transformer 架构**（3.4-3.11）是本章的核心。我们将详细拆解 Transformer 的每个组件：分词与词典向量化、Embedding 与权重共享、位置编码、激活函数、正则化与残差连接、注意力机制、以及混合专家（MoE）网络。这些组件构成了现代大语言模型的技术基座。

**多模态扩展**（3.12）讨论如何在 Transformer 基础上构建处理多种模态的统一模型。

**轻量训练技术**（3.13）介绍 LoRA、量化训练等高效微调方法，使大模型的训练和部署成为可能。

**前沿技术**（3.14）简要介绍线性注意力、状态空间模型（SSM/Mamba）、扩散生成模型等新兴方向。

## 预备知识

阅读本章需要以下数学基础：

- **线性代数**：矩阵乘法、向量内积、特征分解、奇异值分解
- **概率论**：概率分布、条件概率、贝叶斯定理、KL 散度
- **微积分**：多元函数求导、链式法则、梯度计算
- **优化理论**：梯度下降、收敛性分析

同时，读者应熟悉前两章的内容：有监督/无监督学习的基本概念、损失函数与优化器、过拟合与正则化等。

## 符号约定

本章使用以下数学符号：

| 符号 | 含义 |
|------|------|
| $\mathbf{x} \in \mathbb{R}^d$ | $d$ 维输入向量 |
| $\mathbf{W} \in \mathbb{R}^{m \times n}$ | $m \times n$ 维权重矩阵 |
| $\sigma(\cdot)$ | 激活函数 |
| $\mathbf{h}^{(l)}$ | 第 $l$ 层的隐藏状态 |
| $\text{softmax}(\mathbf{z})_i$ | $\frac{e^{z_i}}{\sum_j e^{z_j}}$ |
| $\|\cdot\|_2$ | $L^2$ 范数 |
| $\mathbf{Q}, \mathbf{K}, \mathbf{V}$ | Query、Key、Value 矩阵 |
| $d_k, d_v, d_{\text{model}}$ | Key/Value/模型维度 |

张量维度的书写约定：$(B, L, D)$ 表示批次大小 $B$、序列长度 $L$、特征维度 $D$。
