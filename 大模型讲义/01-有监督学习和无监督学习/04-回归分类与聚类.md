# 1.4 回归、分类与聚类

## 1.4.1 任务类型的统一视角

回归、分类与聚类是机器学习中最基础的三类任务。从函数逼近的角度看，它们可以统一在一个框架下理解：给定输入空间 $\mathcal{X}$ 和输出空间 $\mathcal{Y}$，学习一个映射 $f: \mathcal{X} \rightarrow \mathcal{Y}$。三类任务的区别主要体现在：

1. **输出空间的性质**：连续、离散有限、离散未知
2. **学习范式**：有监督（回归、分类）vs. 无监督（聚类）
3. **评价标准**：预测精度 vs. 结构发现

本节将从数学建模的角度，系统分析这三类任务的形式化定义、损失函数设计、典型算法与评估方法。

## 1.4.2 回归：连续值预测

### 回归问题的数学形式化

回归（Regression）是预测连续值输出的有监督学习任务。给定训练集 $D = \{(x_i, y_i)\}_{i=1}^n$，其中 $x_i \in \mathbb{R}^d$，$y_i \in \mathbb{R}$（或 $\mathbb{R}^m$ 用于多输出回归），目标是学习函数 $f: \mathbb{R}^d \rightarrow \mathbb{R}$。

从概率论的视角，回归可以理解为对条件期望的估计：

$$f^*(x) = \mathbb{E}[Y|X=x] = \int y \cdot p(y|x) \, dy$$

在噪声为加性高斯分布的假设下：

$$y = f(x) + \epsilon, \quad \epsilon \sim \mathcal{N}(0, \sigma^2)$$

最大化似然估计等价于最小化均方误差。

### 损失函数

**均方误差（Mean Squared Error, MSE）**：

$$L_{MSE} = \frac{1}{n} \sum_{i=1}^n (y_i - \hat{y}_i)^2$$

MSE 对应于高斯噪声假设下的最大似然估计。它对异常值敏感，因为误差被平方。

**平均绝对误差（Mean Absolute Error, MAE）**：

$$L_{MAE} = \frac{1}{n} \sum_{i=1}^n |y_i - \hat{y}_i|$$

MAE 对应于拉普拉斯噪声假设。它对异常值的鲁棒性更好，但在零点不可微。

**Huber 损失**：

$$L_\delta(y, \hat{y}) = \begin{cases} \frac{1}{2}(y - \hat{y})^2 & |y - \hat{y}| \leq \delta \\ \delta |y - \hat{y}| - \frac{1}{2}\delta^2 & |y - \hat{y}| > \delta \end{cases}$$

Huber 损失结合了 MSE 和 MAE 的优点：在误差较小时使用平方损失（更平滑），在误差较大时使用绝对值损失（对异常值鲁棒）。

### 线性回归

最简单的回归模型是线性回归（Linear Regression）：

$$f(x) = w^T x + b = \sum_{j=1}^d w_j x^{(j)} + b$$

参数 $w \in \mathbb{R}^d$ 和 $b \in \mathbb{R}$ 通过最小化 MSE 确定。记 $\tilde{X} = [X, \mathbf{1}]$ 为增广数据矩阵，$\tilde{w} = [w; b]$ 为增广权重向量，则：

$$\hat{\tilde{w}} = \arg\min_{\tilde{w}} \|\tilde{X}\tilde{w} - y\|^2$$

通过令梯度为零，可以得到解析解（正规方程）：

$$\hat{\tilde{w}} = (\tilde{X}^T \tilde{X})^{-1} \tilde{X}^T y$$

当特征数 $d$ 大于样本数 $n$ 时，$\tilde{X}^T \tilde{X}$ 不满秩，正规方程没有唯一解。此时需要引入正则化。

### 正则化回归

**岭回归（Ridge Regression）**添加 $L_2$ 正则化：

$$\hat{w} = \arg\min_w \|Xw - y\|^2 + \lambda \|w\|^2$$

解析解为：

$$\hat{w} = (X^T X + \lambda I)^{-1} X^T y$$

$L_2$ 正则化使权重趋向于较小的值，防止过拟合，并保证矩阵可逆。

**LASSO（Least Absolute Shrinkage and Selection Operator）**添加 $L_1$ 正则化：

$$\hat{w} = \arg\min_w \|Xw - y\|^2 + \lambda \|w\|_1$$

$L_1$ 正则化产生稀疏解——许多权重恰好为零。这提供了自动的特征选择能力。

**弹性网（Elastic Net）**结合 $L_1$ 和 $L_2$ 正则化：

$$\hat{w} = \arg\min_w \|Xw - y\|^2 + \lambda_1 \|w\|_1 + \lambda_2 \|w\|^2$$

### 非线性回归

为了拟合非线性关系，可以采用以下策略：

**多项式回归**：将原始特征扩展为多项式特征。例如，对于一维输入 $x$，构造特征 $(1, x, x^2, ..., x^p)$，然后进行线性回归。

**核回归**：利用核技巧（Kernel Trick）隐式地将数据映射到高维空间。核岭回归的预测形式为：

$$f(x) = \sum_{i=1}^n \alpha_i K(x_i, x)$$

其中 $K$ 是核函数（如高斯核 $K(x, x') = \exp(-\|x-x'\|^2 / 2\sigma^2)$）。

**神经网络回归**：使用深度神经网络作为非线性函数近似器，输出层使用线性激活（无激活函数），损失函数使用 MSE。

### 回归的评估指标

**$R^2$ 决定系数**：

$$R^2 = 1 - \frac{\sum_i (y_i - \hat{y}_i)^2}{\sum_i (y_i - \bar{y})^2}$$

$R^2$ 衡量模型解释的方差比例。$R^2 = 1$ 表示完美拟合，$R^2 = 0$ 表示模型的解释能力等同于预测均值。

**均方根误差（RMSE）**：$\sqrt{MSE}$，与原始数据同单位，便于解释。

**平均绝对百分比误差（MAPE）**：$\frac{1}{n}\sum_i \frac{|y_i - \hat{y}_i|}{|y_i|}$，衡量相对误差。

## 1.4.3 分类：离散类别预测

### 分类问题的数学形式化

分类（Classification）是预测离散类别输出的有监督学习任务。给定训练集 $D = \{(x_i, y_i)\}_{i=1}^n$，其中 $x_i \in \mathbb{R}^d$，$y_i \in \{1, 2, ..., K\}$，目标是学习分类器 $f: \mathbb{R}^d \rightarrow \{1, 2, ..., K\}$。

从概率论的视角，分类的目标是估计后验概率 $P(Y=k|X=x)$，然后选择概率最大的类别：

$$\hat{y} = \arg\max_k P(Y=k|X=x)$$

根据贝叶斯定理：

$$P(Y=k|X=x) = \frac{P(X=x|Y=k) P(Y=k)}{P(X=x)}$$

这引出了判别式和生成式两种建模策略（见 1.1 节）。

### 损失函数

**0-1 损失**：

$$L_{0-1}(y, \hat{y}) = \mathbb{I}[y \neq \hat{y}]$$

0-1 损失是最直接的分类损失，但它不连续且非凸，难以优化。实践中使用其替代损失。

**交叉熵损失（Cross-Entropy Loss）**：

对于二分类：

$$L_{CE} = -\frac{1}{n} \sum_{i=1}^n [y_i \log \hat{p}_i + (1-y_i) \log(1-\hat{p}_i)]$$

其中 $\hat{p}_i = P(Y=1|x_i)$ 是预测的正类概率。

对于多分类（$K$ 类）：

$$L_{CE} = -\frac{1}{n} \sum_{i=1}^n \sum_{k=1}^K y_{ik} \log \hat{p}_{ik}$$

其中 $y_{ik}$ 是独热编码的标签。

交叉熵损失是 0-1 损失的凸上界，并且与极大似然估计等价。

**Hinge 损失**：

$$L_{hinge}(y, f(x)) = \max(0, 1 - y \cdot f(x))$$

其中 $y \in \{-1, +1\}$。Hinge 损失是 SVM 使用的损失函数。

### 逻辑回归

逻辑回归（Logistic Regression）是最经典的线性分类模型。对于二分类，模型形式为：

$$P(Y=1|x) = \sigma(w^T x + b) = \frac{1}{1 + e^{-(w^T x + b)}}$$

其中 $\sigma$ 是 sigmoid 函数。

逻辑回归可以从多个角度理解：

**判别式模型视角**：直接建模 $P(Y|X)$，通过最大化条件似然（等价于最小化交叉熵）学习参数。

**广义线性模型视角**：逻辑回归是广义线性模型（GLM）的一个实例，链接函数是 logit 函数：

$$\log \frac{P(Y=1|x)}{P(Y=0|x)} = w^T x + b$$

**决策边界视角**：$w^T x + b = 0$ 定义了特征空间中的一个超平面，将正负类分开。

### 支持向量机

支持向量机（Support Vector Machine, SVM）从几何角度出发，寻找最大间隔的分类超平面。

**硬间隔 SVM**（假设数据线性可分）：

$$\min_{w, b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^T x_i + b) \geq 1, \forall i$$

**软间隔 SVM**（允许分类错误）：

$$\min_{w, b, \xi} \frac{1}{2}\|w\|^2 + C \sum_i \xi_i \quad \text{s.t.} \quad y_i(w^T x_i + b) \geq 1 - \xi_i, \xi_i \geq 0$$

**核 SVM**：通过核技巧处理非线性可分问题。决策函数变为：

$$f(x) = \sum_i \alpha_i y_i K(x_i, x) + b$$

只有支持向量（$\alpha_i > 0$ 的样本）参与预测，这使得 SVM 在测试时高效。

### 神经网络分类

深度神经网络通过多层非线性变换学习复杂的决策边界。对于 $K$ 类分类，输出层通常使用 softmax 函数：

$$P(Y=k|x) = \frac{\exp(z_k)}{\sum_{j=1}^K \exp(z_j)}$$

其中 $z = (z_1, ..., z_K)$ 是最后一个线性层的输出（logits）。

### 分类的评估指标

**准确率（Accuracy）**：

$$\text{Accuracy} = \frac{\text{正确分类样本数}}{\text{总样本数}}$$

在类别平衡的情况下，准确率是合理的指标；但在类别不平衡时可能产生误导。

**精确率、召回率与 F1**：

对于二分类，设 TP（True Positive）、FP（False Positive）、TN（True Negative）、FN（False Negative）：

$$\text{Precision} = \frac{TP}{TP + FP}, \quad \text{Recall} = \frac{TP}{TP + FN}$$

$$F_1 = \frac{2 \cdot \text{Precision} \cdot \text{Recall}}{\text{Precision} + \text{Recall}}$$

精确率关注"预测为正的样本中有多少真正是正的"，召回率关注"正样本中有多少被正确召回"。F1 是两者的调和平均。

**ROC 曲线与 AUC**：

ROC（Receiver Operating Characteristic）曲线绘制不同阈值下的真正率（TPR = Recall）与假正率（FPR = FP/(FP+TN)）。AUC（Area Under Curve）是 ROC 曲线下的面积，衡量分类器区分正负类的能力，不受类别不平衡和阈值选择的影响。

**混淆矩阵**：$K$ 类分类问题的混淆矩阵是 $K \times K$ 矩阵，其中 $C_{ij}$ 表示真实类别为 $i$ 但被预测为 $j$ 的样本数。

## 1.4.4 聚类：数据分组

### 聚类问题的数学形式化

聚类（Clustering）是无监督学习中最常见的任务，目标是将数据划分为若干组，使得组内相似性高、组间相似性低。

形式化地，给定数据集 $D = \{x_1, x_2, ..., x_n\}$，聚类算法寻找一个分配函数 $c: \{1, ..., n\} \rightarrow \{1, ..., K\}$，将每个样本分配到某个簇。聚类数 $K$ 可以是预先指定的，也可以由算法自动确定。

聚类问题的本质困难在于：没有标签，就没有明确的优化目标。不同的聚类算法对"相似性"有不同的定义，对应不同的目标函数。

### K-means 聚类

K-means 是最经典的聚类算法，其目标是最小化簇内平方和（Within-Cluster Sum of Squares, WCSS）：

$$J = \sum_{k=1}^K \sum_{x_i \in C_k} \|x_i - \mu_k\|^2$$

其中 $\mu_k = \frac{1}{|C_k|} \sum_{x_i \in C_k} x_i$ 是第 $k$ 个簇的质心。

**算法流程**：
1. 随机初始化 $K$ 个质心 $\mu_1, ..., \mu_K$
2. **分配步骤**：将每个样本分配到最近的质心：$c(i) = \arg\min_k \|x_i - \mu_k\|$
3. **更新步骤**：重新计算每个簇的质心：$\mu_k = \frac{1}{|C_k|} \sum_{x_i \in C_k} x_i$
4. 重复 2-3 直到收敛

K-means 的每次迭代都单调地降低目标函数，因此必定收敛。但它只能保证收敛到局部最优，结果依赖于初始化。

**K-means++ 初始化**：选择相距较远的初始质心，以获得更好的结果。

**K-means 的局限性**：
- 假设簇是凸的、各向同性的
- 对异常值敏感
- 需要预先指定 $K$
- 只能处理数值特征

### 层次聚类

层次聚类（Hierarchical Clustering）构建一个聚类的层次结构，可以是自底向上（凝聚）或自顶向下（分裂）。

**凝聚层次聚类**：
1. 初始时每个样本自成一簇
2. 计算所有簇对之间的距离
3. 合并距离最近的两个簇
4. 重复 2-3 直到只剩一个簇

簇间距离的定义方式：
- **单链接（Single Linkage）**：最近两点之间的距离
- **全链接（Complete Linkage）**：最远两点之间的距离
- **平均链接（Average Linkage）**：所有点对距离的平均
- **Ward 方法**：合并后 WCSS 增量最小的两个簇

结果用树状图（Dendrogram）表示，可以在任意高度切割得到不同数量的簇。

### 密度聚类

DBSCAN（Density-Based Spatial Clustering of Applications with Noise）基于密度定义簇，能够发现任意形状的簇并自动识别噪声点。

核心概念：
- **ε-邻域**：到点 $p$ 距离不超过 $\epsilon$ 的所有点
- **核心点**：ε-邻域内至少有 MinPts 个点
- **边界点**：不是核心点，但在某个核心点的 ε-邻域内
- **噪声点**：既不是核心点也不是边界点

DBSCAN 从任意核心点出发，递归地将密度可达的点归入同一簇。

优点：不需要预先指定簇数，能处理任意形状的簇，对噪声鲁棒。
缺点：对参数 $\epsilon$ 和 MinPts 敏感，难以处理密度变化大的数据。

### 高斯混合模型

高斯混合模型（Gaussian Mixture Model, GMM）是一种基于概率的聚类方法，假设数据由 $K$ 个高斯分布混合生成：

$$p(x) = \sum_{k=1}^K \pi_k \mathcal{N}(x|\mu_k, \Sigma_k)$$

其中 $\pi_k$ 是混合权重，$\sum_k \pi_k = 1$。

参数通过期望最大化（EM）算法估计：

**E 步**：计算每个样本属于每个高斯分量的后验概率（责任）：

$$\gamma_{ik} = \frac{\pi_k \mathcal{N}(x_i|\mu_k, \Sigma_k)}{\sum_j \pi_j \mathcal{N}(x_i|\mu_j, \Sigma_j)}$$

**M 步**：更新参数：

$$\pi_k = \frac{1}{n}\sum_i \gamma_{ik}, \quad \mu_k = \frac{\sum_i \gamma_{ik} x_i}{\sum_i \gamma_{ik}}, \quad \Sigma_k = \frac{\sum_i \gamma_{ik} (x_i - \mu_k)(x_i - \mu_k)^T}{\sum_i \gamma_{ik}}$$

与 K-means 相比，GMM 提供软聚类（每个样本属于每个簇的概率）而非硬聚类，且能处理椭圆形的簇。

### 聚类的评估

由于没有真实标签，聚类的评估比分类更困难。评估方法分为两类：

**内部指标**（不需要真实标签）：

**轮廓系数（Silhouette Coefficient）**：

$$s(i) = \frac{b(i) - a(i)}{\max\{a(i), b(i)\}}$$

其中 $a(i)$ 是样本 $i$ 到同簇其他点的平均距离，$b(i)$ 是样本 $i$ 到最近其他簇的平均距离。$s(i) \in [-1, 1]$，越接近 1 越好。

**Davies-Bouldin 指数**：衡量簇内离散度与簇间距离的比值，越小越好。

**外部指标**（需要真实标签）：

**调整兰德指数（Adjusted Rand Index, ARI）**：衡量聚类结果与真实标签的一致性，调整了随机情况，取值范围 $[-1, 1]$，1 表示完全一致。

**归一化互信息（Normalized Mutual Information, NMI）**：基于信息论，衡量聚类结果与真实标签的共享信息量。

## 1.4.5 三类任务的对比与联系

| 特性 | 回归 | 分类 | 聚类 |
|------|------|------|------|
| 学习范式 | 有监督 | 有监督 | 无监督 |
| 输出类型 | 连续值 | 离散类别（已知） | 离散类别（未知） |
| 目标 | 预测数值 | 预测类别 | 发现分组 |
| 损失函数 | MSE, MAE | 交叉熵, Hinge | WCSS, 似然 |
| 评估方式 | $R^2$, RMSE | 准确率, F1, AUC | 轮廓系数, ARI |

三者之间存在密切联系：

- **分类可视为离散化的回归**：预测类别概率后选择最大概率的类别
- **聚类可为分类提供标签**：先聚类，再将聚类标签作为伪标签进行有监督学习
- **回归可用于聚类评估**：如预测聚类数 $K$
